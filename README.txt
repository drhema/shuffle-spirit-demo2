Next.js App
===========

This is a Next.js project bootstrapped with create-next-app.
(https://nextjs.org/)


GETTING STARTED
---------------

First, install the dependencies:

    npm install
    # or
    yarn install
    # or
    pnpm install

Then, run the development server:

    npm run dev
    # or
    yarn dev
    # or
    pnpm dev

Open http://localhost:3000 in your browser to see the result.
